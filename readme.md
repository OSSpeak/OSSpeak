Voice control language/IDE.

<h2>Installation</h2>
Osspeak uses Python 3.6.
<h4>From source</h4>

```
git clone https://github.com/osspeak/osspeak.git
python -m venv osspeak
cd osspeak
scripts\activate
pip install -r requirements.txt
```
